========================================
萬用轉檔工具 - 攜帶版使用說明
========================================

【已完成的工作】
✅ 建立完整的桌面 GUI 應用程式
✅ 打包成可攜帶版本 (Universal-Converter-Portable.zip)
✅ 檔案大小：約 308 MB

========================================
【如何攜帶到其他電腦】
========================================

方法 1：使用 USB 隨身碟
------------------------
1. 將 Universal-Converter-Portable.zip 複製到 USB
2. 到其他電腦插入 USB
3. 複製 ZIP 檔案到電腦
4. 解壓縮（對檔案按右鍵 → 解壓縮全部）
5. 進入 Universal-Converter-win32-x64 資料夾
6. 雙擊執行 Universal-Converter.exe

方法 2：使用雲端硬碟
------------------------
1. 上傳 Universal-Converter-Portable.zip 到：
   - Google Drive
   - OneDrive
   - Dropbox
   等雲端服務
2. 在其他電腦下載
3. 解壓縮並執行

方法 3：透過網路傳輸
------------------------
1. 使用 Email 附件（如果檔案太大，使用雲端連結）
2. 使用 WeTransfer 等檔案傳輸服務
3. 使用區域網路共享

========================================
【在其他電腦執行】
========================================

1. 解壓縮 Universal-Converter-Portable.zip
2. 開啟 Universal-Converter-win32-x64 資料夾
3. 雙擊 Universal-Converter.exe
4. 應用程式會開啟一個獨立視窗
5. 不需要網路連線
6. 不需要安裝任何軟體

========================================
【系統需求】
========================================
✅ Windows 10 或 Windows 11
✅ 64 位元系統
✅ 約 500 MB 可用硬碟空間（解壓縮後）
❌ 不需要 Node.js
❌ 不需要安裝 FFmpeg
❌ 不需要其他依賴軟體

========================================
【檔案位置】
========================================
壓縮檔案：
C:\Users\user\OneDrive\tuf18\universal-converter\Universal-Converter-Portable.zip

原始應用程式：

========================================
【使用方式】
========================================
1. 啟動程式後會看到轉檔工具視窗
2. 拖放檔案或點擊選擇檔案
3. 選擇輸出格式
4. 點擊轉換
5. 下載轉換後的檔案

========================================
【支援的檔案格式】
========================================
圖片：JPG, PNG, GIF, WebP, BMP
影片：MP4, AVI, MOV, MKV, WebM
音訊：MP3, WAV, FLAC, AAC, OGG
文檔：PDF, DOCX, TXT, XLSX, CSV

========================================
【疑難排解】
========================================

問題：Windows 保護電腦警告
解決：點擊「更多資訊」→「仍要執行」

問題：防毒軟體警告
解決：這是誤報，可以信任此程式

問題：程式無法啟動
解決：確認是 Windows 10/11 64位元系統

問題：轉換失敗
解決：檢查檔案格式是否支援
